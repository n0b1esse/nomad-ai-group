# Nomad AI Group — Frontend

Premium B2B Investment Platform. Next.js (App Router), Tailwind CSS, Framer Motion, Lucide Icons, Recharts.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Страницы

- **/** — Главная: Hero, AUM-виджет, блок «Наши стратегии», карточки вертикалей (GenAI, Quantum, Biotech).
- **/about** — Философия Nomad, блок «Команда» (имена, роли, ссылки на профили).
- **/portal** — Инвестор-портал: форма входа и dashboard (график доходности Recharts, позиции, кнопка связи с партнёром).
- **/analytics** — Insights: список статей/отчётов (Date | Title | Category).

## Сборка

```bash
npm run build
npm start
```

Тема: Dark Premium (Obsidian #050505, Slate, Silver). Анимации при скролле (Framer Motion), микро-интеракции кнопок (glass-shine), адаптивная вёрстка.
